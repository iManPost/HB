ต้องเปิดไฟล์นี้ในเบราว์เซอร์โดยตรง (ไม่ใช่ผ่านเว็บอื่น)
พร้อมใช้งาน
⚠️ วิธีแก้บล็อก CORS:
ไฟล์นี้ต้องเปิดใน Chrome/Edge โดยตรง และต้องติดตั้ง extension ก่อน:
→ CORS Unblock (แนะนำ) หรือ Allow CORS: Access-Control-Allow-Origin
→ เปิด extension ให้ active (ไอคอนเขียว) แล้วรีโหลดหน้านี้ แล้วกดเล่น
วิธีใช้ทีละขั้น:
1. ดาวน์โหลดไฟล์ hbo-player.html นี้ไว้ในเครื่อง
2. ติดตั้ง extension CORS Unblock ใน Chrome
3. เปิดไฟล์โดยลาก .html เข้า Chrome หรือ File → Open
4. กด icon extension ให้เป็น "ON" (สีเขียว)
5. เลือกช่องแล้วกด เล่น
